使用Git Bush对Github进行同步

Sep 1, 2017 | Configuration Tutorial | 阅读

GitHub部分：

1. 注册GitHub账号

step 1：打开GitHub主页GitHub

step 2：填写用户名，邮箱，密码，点击Sign up for GitHub

step 3：不勾选任何选项，直接点击Continue

step 4：不勾选任何选项，直接点击Submit旁边的skip this step

step 5：点击发送来的验证邮件中的确认链接，验证你的邮箱

step 6：进入你的空主页

2. 新建一个空的项目

step 1：点击空主页上的Start a project开始新建一个新的代码仓库

step 2：只在1处填写仓库名称（自己取一个）这里我新建的代码仓库叫做code，2处是对这个仓库的描述，可以写也可以不写

step 3：点击下方的Creat repository完成新建代码仓库

step 4：在代码仓库为空的情况下，会默认显示这个远程管理方式提示界面

Git Bush部分：

1.下载Git Bush客户端

step 1：打开Git下载页Git Download

step 2：选择对应版本进行下载，这里下载windows版本

step 3：一般有32位和64位程序可供选择，这里选择64位进行下载

2. 安装Git Bush客户端

step 1：双击安装包

step 2：根据提示依次点击next直到完成安装，期间并不需要对任何选项做更改

step 3：等待安装完成

step 4：安装完成，去掉View Release Notes前面的对勾，点击Finish

3.初始化Git Bush客户端

step 1：在电脑里新建一个文件夹作为存放代码的位置

step 2：进入这个文件夹，点击右键看到Git Bush选项，左键点击进入
